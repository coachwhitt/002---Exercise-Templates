# CoachWhitt Brand Reference Context

This folder contains all essential brand reference materials for the CoachWhitt coaching business. These documents and assets define the complete brand identity, visual specifications, and messaging framework.

This reference context is:
- **Portable:** Move to any global location for use across multiple projects
- **Complete:** All essential brand elements in one place

- **Self-Contained:** All file references use relative paths within this folder

## Common Questions

**Q: Which logo file should I use?**
A: Use `05-logo-main.svg` for 90% of applications (web, print, merchandise). Only use PNG versions when SVG is not supported (email signatures, some social media).

**Q: Where are the brand colors?**
A: Complete color specifications are in `04-visual-brand-guide.md` ‚Üí "Official Brand Colors" section. Includes Hex, RGB, CMYK, and Pantone.

**Q: Where are the Pantone colors for print?**
A: `04-visual-brand-guide.md` ‚Üí "Color Usage for Print/Merchandise" section. Black 6 C, 7527 C, 2028 C.

**Q: What font do we use?**
A: Montserrat (Google Fonts). Full typography specifications in `04-visual-brand-guide.md` ‚Üí "Typography" section.

**Q: Where's the brand story?**
A: `03-brand-outline.md` ‚Üí Section 6: "Brand Story" (2023 European Championship final).

**Q: How do I write content in the CoachWhitt voice?**
A: Start with `01-universal-principles.md` for the 3 Cs framework, then `02-coachwhitt-framework.md` for content pillars.
