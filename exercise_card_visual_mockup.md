## Color Legend

**Visual Representation:**
- `░░░` = Deep Charcoal (#1E1F22) - Primary Background
- `┌──┐` = Secondary Background (#25262A) - Card sections
- `┃──┃` = Red left border accent (#F55139)
- `▓▓▓` = Footer Background (#15161A)
- `▓▓▓` (in diagrams) = Red muscle highlighting (#F55139)
- `███` (in diagrams) = Secondary muscle highlighting (50% opacity #F55139)

**Text Colors:**
- Primary headings: Off-White (#EBE7D9)
- Body text: Off-White (#EBE7D9)
- Labels: Muted (#B8B3A8)
- "COACHING POINTS" heading: Accent Red (#F55139)

## Asset Requirements Summary

For each exercise card generation, the following assets are required:

### Required Images (Per Exercise)

1. **Male Front Anatomy Diagram** (SVG preferred, PNG fallback)
   - Size: 200 × 350px at 300 DPI
   - Primary muscles highlighted in #F55139 (100% opacity)
   - Secondary muscles highlighted in #F55139 (50% opacity)

2. **Male Rear Anatomy Diagram** (SVG preferred, PNG fallback)
   - Same specifications as front

3. **Female Front Anatomy Diagram** (SVG preferred, PNG fallback)
   - Same specifications as male front

4. **Female Rear Anatomy Diagram** (SVG preferred, PNG fallback)
   - Same specifications as male rear

5. **Exercise Illustration - Male** (SVG preferred)
   - Size: 300 × 200px at 300 DPI
   - Shows 2 phases of movement (start/end or bottom/top)
   - Matches anatomical diagram style

6. **Exercise Illustration - Female** (SVG preferred)
   - Same specifications as male illustration

### Optional Video Assets

7. **Instructional Video - Male** (MP4, H.264 codec)
   - Resolution: 1080p minimum
   - Length: 15-30 seconds
   - Shows full exercise with muscle highlighting overlay

8. **Instructional Video - Female** (MP4, H.264 codec)
   - Same specifications as male video

## Generation Workflow Example

### Input: JSON Data File

```json
{
  "exerciseName": "Barbell Bench Press",
  "movementType": "Compound",
  "primaryMuscles": ["Pectoralis Major"],
  "equipment": ["Barbell", "Flat Bench"],
  ...
}
```

### Process:
1. Load JSON data
2. Select appropriate anatomical diagrams (male or female)
3. Apply CoachWhitt template
4. Populate all content sections
5. Render to PDF at 300 DPI
6. Save with proper naming convention

### Output: PDF Exercise Card

- File: `Barbell_Bench_Press_Male_Card.pdf`
- Size: 1748 × 2480 px (A5 at 300 DPI)
- Pages: 1
- File size target: < 2MB

## Quality Checklist

Before approving any exercise card:

- [ ] Exercise name is clear and prominent
- [ ] Movement type badge is visible
- [ ] CoachWhitt logo is present in header
- [ ] Both anatomical diagrams show correct muscles
- [ ] Primary muscles highlighted at full red opacity
- [ ] Secondary muscles highlighted at 50% red opacity
- [ ] Muscle labels match diagram highlights
- [ ] Equipment list is accurate
- [ ] Instructions are numbered 1-6 (or appropriate count)
- [ ] Coaching points are 3-5 bullets
- [ ] Exercise illustration shows correct form
- [ ] Tracking fields are present and editable
- [ ] Footer includes tagline and page number
- [ ] All text is legible at print size
- [ ] Colors match brand guide exactly
- [ ] Spacing is consistent throughout
- [ ] File is named correctly
- [ ] Resolution is 300 DPI for print version

## Important Notes

**Database vs. Client-Facing Content:**

The exercise database contains additional fields not shown on client cards:
- **Sports Tags:** Used by trainer for sport-specific program selection (not on cards)
- **Scientific References:** Used by trainer for validation and review (not on cards)
- **Difficulty Level:** Used for program planning (not on cards)
- **Muscle Group Category:** Used for database organization (not on cards)

These fields remain in the database for trainer use but do not appear on the client-facing exercise cards.
